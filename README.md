# 🌤️ Weather App

A web application built with HTML, CSS, and JavaScript that provides weather information for cities around the world.
The app retrieves live weather data through a weather API and displays a weekly forecast in a clean, user-friendly interface.

## 🚀 Features

* 🔍 Search weather by city name
* 📅 Displays weekly weather forecast
* 🌡️ Shows temperature details
* ☁️ Weather conditions with icons
* 💨 Wind speed information
* 💧 Humidity percentage

## 🛠️ Technologies Used

* HTML
* CSS
* Vanilla JavaScript 
* Weather API
